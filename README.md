# mbr88
An OS independent 8088 safe MBR for dual booting ELKS, FreeDOS or other OS.

This boot loader ignores the active partition flag and let's the user 
select the partition (or floppy) to boot.

Written by Claude Code under the direction of C. Piker.  Verified on a 
Leading Edge Model D with two floppy drives and a four partition hard
disk with both ELKS Linux and FreeDos installed.
